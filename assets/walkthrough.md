# Walkthrough - Woodlawn HS Class of 2006 Reunion Landing Page

I have completed the high-conversion, nostalgic landing page for the Woodlawn High School Class of 2006 20-Year Reunion. The site captures the "Sound of '06" vibe by blending mid-2000s Baltimore urban culture with premium modern design.

## Key Features

- **Hero Section:** A bold, high-impact entrance with the "Sound of '06" headline, integrated event details, and a high-conversion "Get Tickets" CTA.
- **Experience Zones:** An interactive grid showcasing the Tech Station, 360 Photo Booth, and Memory Wall.
- **Ticket Tiers:** A glassmorphism-styled pricing table featuring Alumni, Plus 1, and Legacy tiers.
- **Alumni Vendor Call:** A dedicated section to empower business owners within the Class of '06.
- **Premium Aesthetics:** 
    - Custom palette: Razr Silver, Carbon Black, 92Q Orange, and Sidekick Blue.
    - Advanced CSS: Glassmorphism, HSL-based design system, and smooth scroll animations.
    - Responsive: Mobile-first architecture for alumni on the go.
